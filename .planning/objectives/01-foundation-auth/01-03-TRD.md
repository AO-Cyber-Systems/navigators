<objective>
Create the domain schema for turfs, teams, and voter access audit logging. Build the turf-scoped data filtering utility that enforces Navigator sees own turfs, Super Navigator sees team turfs, Admin sees all. Build the audit log viewer for admins.

Purpose: This completes the data access control story. Eden RBAC handles "can this user access this feature?" but turf scoping handles "which data within that feature can they see?" The audit trail ensures compliance with Maine voter data law (Title 21-A Section 196-A).
Output: Domain tables, turf-scoped query utility, audit logging utility, admin audit viewer endpoint.
</objective>

// Service layer enforces turf scoping AFTER eden RBAC allows the request.
// This is NOT an interceptor -- it's called in every voter data query.
func (s *VoterService) ListVoters(ctx context.Context, userID, companyID uuid.UUID) ([]Voter, error) {
    claims := server.ClaimsFromContext(ctx)
    switch {
    case claims.RoleLevel >= int(rbac.RoleLevelAdmin):
        return s.store.ListAllVoters(ctx, companyID)
    case claims.RoleLevel >= int(rbac.RoleLevelManager):
        turfIDs, _ := s.teamStore.GetTeamTurfIDs(ctx, userID)
        return s.store.ListVotersByTurfs(ctx, turfIDs)
    default:
        turfIDs, _ := s.turfStore.GetUserTurfIDs(ctx, userID)
        return s.store.ListVotersByTurfs(ctx, turfIDs)
    }
}

<anti_patterns>
- DO NOT put turf-scoping logic in interceptors. Interceptors handle permission checks. Turf scoping is data filtering and belongs in the service/query layer.
- DO NOT use eden company hierarchy for team hierarchy. team_assignments is a simpler, dedicated table.
- DO NOT hardcode role level numbers. Use rbac.RoleLevelMember, rbac.RoleLevelManager, rbac.RoleLevelAdmin constants.
- DO NOT query all voters and filter in Go. Push turf filtering down to SQL with WHERE turf_id IN (...).
- DO NOT skip voter_access_log for any voter data access. Every view, edit, export, and search must be logged.
</anti_patterns>

1. **internal/navigators/turf_scope.go**: Reusable turf-scoping utility.

   ```go
   package navigators

   import (
       "context"
       "github.com/aocybersystems/eden-platform-go/platform/rbac"
       "github.com/aocybersystems/eden-platform-go/platform/server"
       "github.com/google/uuid"
   )

   // ScopeType indicates the data scope for a user.
   type ScopeType int
   const (
       ScopeOwn  ScopeType = iota // Navigator: own turfs only
       ScopeTeam                   // Super Navigator: team's turfs
       ScopeAll                    // Admin: all data
   )

   // TurfScope represents a resolved data scope for a user.
   type TurfScope struct {
       Type    ScopeType
       TurfIDs []uuid.UUID // populated for ScopeOwn and ScopeTeam; nil for ScopeAll
       UserID  uuid.UUID
   }

   // TurfScopedFilter resolves what data a user can see based on their role and assignments.
   type TurfScopedFilter struct {
       queries *db.Queries // sqlc-generated queries
   }

   func NewTurfScopedFilter(queries *db.Queries) *TurfScopedFilter {
       return &TurfScopedFilter{queries: queries}
   }

   // ResolveScope determines the data scope for the current user from context.
   func (f *TurfScopedFilter) ResolveScope(ctx context.Context) (*TurfScope, error) {
       claims := server.ClaimsFromContext(ctx)
       if claims == nil {
           return nil, fmt.Errorf("no claims in context")
       }

       userID, _ := uuid.Parse(claims.UserID)

       switch {
       case claims.RoleLevel >= int(rbac.RoleLevelAdmin):
           return &TurfScope{Type: ScopeAll, UserID: userID}, nil

       case claims.RoleLevel >= int(rbac.RoleLevelManager):
           // Super Navigator: get all turfs from their team
           turfIDs, err := f.queries.GetTeamTurfIDs(ctx, userID)
           if err != nil {
               return nil, fmt.Errorf("get team turf IDs: %w", err)
           }
           return &TurfScope{Type: ScopeTeam, TurfIDs: turfIDs, UserID: userID}, nil

       default:
           // Navigator: get own assigned turfs
           turfIDs, err := f.queries.GetUserTurfIDs(ctx, userID)
           if err != nil {
               return nil, fmt.Errorf("get user turf IDs: %w", err)
           }
           return &TurfScope{Type: ScopeOwn, TurfIDs: turfIDs, UserID: userID}, nil
       }
   }
   ```

   # CRITICAL: This utility is called by every voter data service. It does NOT query voters itself.
   # It returns the TurfScope which the caller uses to constrain their own queries.
   # Example usage: scope, _ := filter.ResolveScope(ctx); if scope.Type == ScopeAll { queryAll() } else { queryByTurfs(scope.TurfIDs) }
