<anti_patterns>
- DO NOT load timeline data eagerly all at once -- use Drift reactive streams (watch*) so updates appear in real-time
- DO NOT build a custom timeline widget from scratch -- use EdenTimeline from eden-ui-flutter
- DO NOT put the notes tab inline with other profile sections in a single scroll -- use TabBarView for clear separation
- DO NOT show notes the user shouldn't see -- visibility is enforced server-side during pull sync, but also filter client-side as defense-in-depth
</anti_patterns>

<error_recovery>
- If EdenTimeline doesn't exist or has a different API, build a simple ListView with leading icon, title, subtitle, and trailing timestamp
- If Rx.combineLatest is not available (no rxdart), use StreamZip or manual stream combination with StreamController
- If voter has hundreds of timeline entries, add pagination (load 50, "Show More" button)
</error_recovery>
