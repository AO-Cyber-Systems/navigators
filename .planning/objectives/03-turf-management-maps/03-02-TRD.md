<anti_patterns>
- DO NOT load all voters at once. Use viewport bounding box queries + pagination. Only load voters visible in current map bounds.
- DO NOT use supercluster package (incompatible with flutter_map 8). Use flutter_map_marker_cluster 8.2.2.
- DO NOT use flutter_map_heatmap (incompatible with flutter_map 8). Heat maps are in TRD 03-03 via custom Canvas.
- DO NOT store polygon as array of floats in proto -- use GeoJSON string. Standard interchange format.
- DO NOT skip userAgentPackageName on TileLayer. OSM requires proper User-Agent identification.
</anti_patterns>

<error_recovery>
- If flutter_map_line_editor doesn't work with flutter_map 8.2.2: fall back to manual tap-to-add polygon drawing using MapOptions.onTap to collect LatLng points, render with PolygonLayer. The PolyEditor from line_editor is convenience, not essential.
- If flutter_map_marker_cluster has rendering issues: verify markers list isn't empty, check that MarkerClusterLayerWidget is a direct child of FlutterMap (not nested in another layer).
- If GeoJSON parsing fails on boundary response: check that server returns valid GeoJSON with 'type': 'Polygon' and 'coordinates' as nested arrays. Verify winding order is CCW.
</error_recovery>

<research_context>
From 03-RESEARCH.md:
- flutter_map 8.2.2 for map rendering. FlutterMap widget with children: [TileLayer, PolygonLayer, MarkerClusterLayerWidget]
- flutter_map_marker_cluster 8.2.2: MarkerClusterLayerWidget with maxClusterRadius, markers list, builder for cluster circles
- flutter_map_line_editor 8.0.0 + flutter_map_dragmarker 8.0.0: PolyEditor with pointIcon, intermediateIcon, addClosePathMarker: true, callbackRefresh
- latlong2 ^0.9.1 for LatLng coordinate type
- OSM tiles: urlTemplate 'https://tile.openstreetmap.org/{z}/{x}/{y}.png', userAgentPackageName: 'com.mainegop.navigators'
- GeoJSON to LatLng conversion: parse coordinates array [[lng, lat], ...], map to LatLng(lat, lng). Note: GeoJSON is [longitude, latitude], LatLng is (latitude, longitude).
- Initial map center: Augusta, Maine = LatLng(44.3106, -69.7795), zoom 8
</research_context>

<gotchas>
- GeoJSON coordinate order is [longitude, latitude]. LatLng constructor is (latitude, longitude). Swapping these is the most common mapping bug.
- flutter_map_line_editor requires flutter_map_dragmarker as a dependency. Both must be ^8.0.0 for flutter_map 8 compatibility.
- PolygonLayer polygons must have at least 3 points to render. Guard against empty/incomplete polygons.
- MarkerClusterLayerWidget must be a direct child of FlutterMap (in the children list), not nested inside another widget.
- OSM tile usage policy: set userAgentPackageName, respect rate limits. For development this is fine; production may need commercial tile provider.
</gotchas>
